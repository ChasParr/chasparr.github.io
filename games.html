Games

Transimian

Group size: 5
Dates: Spring 2018
Role: game mechanics, level design
Summary: A platformer made for Global Game Jam 2018, which had a theme of "Transmission". The player controls a test chimpanzee named Edison, and must navigate test chambers using a teleportation gun.

Sheepherd

Group size: 5
Dates: Fall 2017
Role: game mechanics, art, flocking, real time multiplayer
Summary: A multiplayer online sheep herding game. Users play as a sheep dog, who must herd sheep to their shepherd.

Community Garden

Group size: Solo
Dates: Spring 2017
Role: game mechanics, art, server-client, real time multiplayer
Summary: A cooperative online gardening game. Users can plant seeds, which can be watered by any user in the room. Watering plants makes them grow over time and also gives users "Karma" which is used as a currency. In a final version of the game Karma would also be purchasable through microtransactions, represented here by the buttons at the bottom.

The Opportune King

Group size: 5
Dates: Spring 2016
Role: core mechanics, card development, balance, playtesting
Summary: A tabletop card game where players compete for influence in an attempt to place themselves on the throne. Each player is assigned a secret role at the beginning of the game that describes their goals, which they can work towards by playing cards.

Asteroids

Group size: Solo
Dates: Fall 2015
Role: physics engine, game mechanics
Summary: A variation of the retro game Asteroids made using Processing. Changes from the original game include different ammo types, aiming with the mouse, and accuracy tracking.
